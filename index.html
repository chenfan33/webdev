User problem: As an important requirement for applying graduate school, GRE General test is very popular in China. It's hard for people to register for an exam slot since people don't know when and where will the exam slots be released. This website provides test takers with a place that they are able to see when and where are those slots avaible so that they don't need to check available slots in each location and time. Hopefully, can help test takes register for a slot automatically.

Users: It is designed for those test takers in the area that GRE is very popular and hard to register.

Comparison: There are some organizations that help them register but the service fee is high.
